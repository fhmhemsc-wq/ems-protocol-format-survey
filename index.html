Google Sheets Setup — one time, ~5 minutes

1. Go to sheets.google.com → create a new sheet named EMS Protocol Survey

2. Click Extensions → Apps Script → delete all existing code → paste the code from apps-script.gs

3. Click Deploy → New deployment
  • Type: Web app
  • Execute as: Me
  • Who has access: Anyone
  • Click Deploy → authorize when prompted

4. Copy the Web App URL → paste it in the field on the left → Save → Test

⚠ Important: After any code change, always create a New deployment (not update) to get a fresh URL.
